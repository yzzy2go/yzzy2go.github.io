Web Automation

I automated the sorting of user-reported bugs using Selenium and Javascript. The automation would start after logging into the web-app then continue until all bugs were placed into the backlog or their respective projects.

Once finished, a total count of each category would be output to the terminal for a quick overview of the bug trends.

Regression Testing

Depending on the release cycle I would conduct regression testing every 1-2 weeks. In my runs, I ensured that each feature was working as expected, with particular attention to the areas where code had been changed.

Found bugs were reported to the devs. If the bug was particularly nasty, I would capture device logs to help get to the root of the problem.

Test Case Creation

To ensure all flows of new or upcoming features would be covered, I created tests for their use cases and added them to a repository. Analysis was completed from mockups, conversations with the designers, or freestyle testing on the dev environment.

Aside from the initial creation, I also kept all test cases up-to-date by making adjustments when new behaviour was implemented.
