Player Controls

I added controls to allow the player to jump and increase their fall speed after pressing the down arrow key. This is the main mechanic of the game and it was achieved by adding velocity to the player's Rigidbody component on a key press if the character was grounded.

Colliders & Obstacles

2D box colliders were used around the player, obstacles and ground to allow for interactions between the parts. If an obstacle came into contact with the "Player" tag, the game would end. If the player was able to successfully avoid the obstacles, the obstacles would run into a collider off-screen which would destroy their gameObject.

Navigation

The main menu had its own scene which would load the Game scene when the start button was pressed. When the player died, the game over panel would appear and it showed their final score along with a button to restart the game.

Extras

To make the game feel more complete I added simple animations to the character which used trigger events to check if they were currently running or jumping. Buttons would also change their state depending on if they were being hovered or pressed. Finally, simple background music was added as an audio component to finish things off.
